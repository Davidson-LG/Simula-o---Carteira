# 📈 Fronteira Eficiente — Renda Fixa

Simulador da Fronteira Eficiente de Markowitz para portfólios de Renda Fixa, incluindo Fundos Imobiliários.

## 🌐 Acesso

Acesse diretamente pelo GitHub Pages:  
👉 **[https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO](https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO)**

## 📋 Funcionalidades

- Simulação de Monte Carlo para geração da Fronteira Eficiente
- 6 estratégias de Renda Fixa:
  - RF — Liquidez
  - RF — IMA Ativo
  - RF — IMA Passivo
  - RF — Ativa CDI
  - RF — Crédito Privado
  - Fundos Imobiliários
- Adição e remoção de ativos por estratégia
- Configuração de taxa livre de risco, IPCA e CDI
- Gráfico interativo com Fronteira Eficiente, Linha do Mercado de Capitais (LMC)
- Portfólio de Variância Mínima e Máximo Índice de Sharpe destacados
- Tabela completa com retorno real, % do CDI e Sharpe
- Resumo por estratégia

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `fronteira-eficiente`)
2. Faça upload do arquivo `index.html`
3. Vá em **Settings → Pages**
4. Em **Source**, selecione `Deploy from a branch`
5. Escolha a branch `main` e pasta `/ (root)`
6. Clique em **Save**
7. Aguarde alguns minutos e acesse a URL exibida
